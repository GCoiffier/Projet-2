
pour compiler, lancer
make

pour executer le programme, lancer
./calc -expr
ou
./calc -fact

dans le premier cas, entrer ensuite une expression arithmetique (avec
juste + et *)

dans le second, entrer un entier, dont le programme calculera la
factorielle

regardez la fin du fichier main.ml pour voir comment on recupere
l'option passee en ligne de commande
